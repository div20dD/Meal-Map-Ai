# MealMap AI - UI Flow

The application is designed with a simple and user-friendly flow:

1. Home Screen
- Scan refrigerator option
- Input: age, BMI, budget, health goal

2. Meal Planning Screen
- Displays 7-day personalized meal plan
- Breakfast, lunch, and dinner suggestions

3. Grocery List Screen
- Budget-based shopping list
- Suggestions for new food items

4. Future Health Screen
- Comparison between current eating habits
- Improved lifestyle path if recommendations are followed
